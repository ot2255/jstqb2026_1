JSTQB CTFL 4.0 模擬試験（本番＋α）40問 パック

■ 使い方
1) zipを展開
2) index.html をブラウザで開く（ダブルクリックOK）
3) 開始 → 解答 → 提出
4) 提出後に TSV（Excel用）をダウンロード

■ 仕様
- CTFL 4.0 ベース（本番比率寄せ）
- 40問 / 4択 / 1問1正解 / 合格 65%（26問）
- タイマーなし（本番感の読解に集中）
- 回答はブラウザ内に自動保存（localStorage）
- 図は img/ にPNGとして同梱（新規作成）

■ 途中保存
- 同じブラウザなら自動復元
- 端末を変える場合は「途中をJSONで保存」→「復元(JSON)」を使用
